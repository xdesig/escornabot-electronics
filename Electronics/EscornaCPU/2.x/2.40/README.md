## Escorna CPU 2.4
 It is a pcb that integrates an ATmega328P, a step-up (to use 1.2V batteries) and the rest of the electronics needed to build an Escornabot.<br>
 Arduino uno, Arduino Nano Compatible or minicore ATmega
